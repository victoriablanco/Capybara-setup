# Setup

* Clone this repo into workspace
* cd into this repo
* `bundle`
* `rake db:create db:migrate db:seed`
* `rails s -p 3012`
* Open http://localhost:3012

## Find the bug!

Once you've added RSpec and capybara, spec the entire crud flow, and find and fix the bug.
